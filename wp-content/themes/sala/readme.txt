Install Steps
--------------
Step 1: Go to Appearance > Add New Themes, then click Upload button

Step 2: Go to browse, then select the sala.zip file

Step 3: Click "Install Now" to install Sala

Step 4: Activate the newly installed Sala by going to Appearance > Themes, find Sala and click on "Activate" button

Step 5: Once Sala is activated, you will be redirected to Sala Welcome Screen. Switch to Plugins tab to install plugins.

Step 6: click "Install" to begin installing plugins. After that, you will have to activate plugins.
